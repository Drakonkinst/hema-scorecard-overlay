How to Use

  1. See the User Guide for details on how to set up this overlay.
  2. Press Open Overlay and use that link as the browser source in OBS.
  3. Press Configure Overlay and use that link as a custom dock in OBS.
  4. If you're having trouble connecting to the HEMA Scorecard API, make sure to set up a CORS workaround as described in the User Guide.
