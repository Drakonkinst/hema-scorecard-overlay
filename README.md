# HEMA Scorecard Overlay

An OBS overlay from HEMA Scorecard.

## Setup

Requires:

* [Node.js](https://nodejs.org/en/download)
  * Downloading through the Windows Installer (.msi) is easiest

```bash
npm install
npm run build
```

Grant esbuild.exe runtime permissions, if prompted.

## Running the Tool

```bash
npm start
```
